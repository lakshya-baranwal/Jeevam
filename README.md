## The Problem

A semi-literate farmer stands in a monsoon field with low internet connectivity. It is 6am. His tomato leaves have turned yellow overnight. His loan repayment is next month.

Today, he calls a relative, waits for a government helpline, or вҖ” worse вҖ” buys whatever the agrochemical dealer recommends. There is no organic expert. No instant diagnosis. No voice-native tool built for him.

**JEEVAM closes that gap in one tap.**

---

## In Action

```
Farmer taps once вҶ’ speaks crop problem in Hindi or English (We can add regional languages as well)
вҶ’ Organic remedy in 15 seconds вҶ’ Works even if internet drops halfway
```

| Scenario | Response |
|:---|:---|
| Tomato leaves yellowing at 6am, 2G internet | Voice diagnosis + neem spray recipe in Hindi |
| "Monocrotophos kaunsi matra mein daalun?" | Refuses chemical, offers organic alternative |
| Internet drops mid-session | Offline cache kicks in transparently |
| Farmer mentions financial distress | Kisan Helpline (1800-180-1551) surfaced immediately |

## Prompt Design

The system prompt is the brain of Jeevam. Every word is intentional.

```
ORGANIC-ONLY GUARDRAIL    вҶ’ refuses all chemicals, always redirects to organic
BILINGUAL DETECTION       вҶ’ Hindi / English / Hinglish auto-detected from input
STRUCTURED OUTPUT         вҶ’ Problem вҶ’ Confirm вҶ’ Treatment вҶ’ Prevention вҶ’ Timing
EMERGENCY PROTOCOL        вҶ’ distress signals surface Kisan Helpline immediately
HOUSEHOLD QUANTITIES      вҶ’ cups, liters, handfuls вҖ” never grams or ML or mg
FARMER'S VOCABULARY       вҶ’ "khet" not "agricultural land", "keeda" not "pest"
```

See [`llm.py`](llm.py) for the full system prompt.

## Offline Strategy

Core disease KB (30 patterns) + education KB (15 techniques) are bundled as local JSON files (~2.5 MB total). When the LLM is unreachable:

```
Farmer speaks
вҶ’ Whisper transcribes locally
вҶ’ Keyword match in local JSON
вҶ’ Cached answer returned
вҶ’ gTTS generates audio
вҶ’ Farmer hears the answer
```

The offline fallback is transparent вҖ” the farmer is told:
> *"Abhi internet nahi hai, isliye stored knowledge se jawab de raha hoon."*
